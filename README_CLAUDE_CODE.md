# Instructions pour Claude Code

## Fichier de brief à consulter

Le fichier **`METRANE_CONSEIL_BRIEF.md`** à la racine de ce projet contient toutes les spécifications pour créer la section METRANE Conseil.

---

## Ta mission

1. **Lis le fichier `METRANE_CONSEIL_BRIEF.md`** pour comprendre le contenu et les spécifications

2. **Analyse la page `products.html`** existante pour comprendre sa structure HTML/CSS

3. **Modifie `products.html`** pour :
   - Ajouter un switch/toggle permettant de basculer entre "METRANE Tools" (contenu actuel) et "METRANE Conseil" (nouveau contenu)
   - METRANE Tools doit être actif par défaut
   - Créer tout le contenu de la section METRANE Conseil tel que décrit dans le brief
   - Utiliser la palette couleurs banque/conseil (bleu marine, or) pour la section Conseil
   - Garder les couleurs actuelles pour la section Tools

4. **Assure-toi que** :
   - Le switch fonctionne sans rechargement de page
   - L'URL `products.html#conseil` affiche directement la section Conseil
   - Le responsive est bien géré (mobile, tablet, desktop)
   - Le style s'intègre harmonieusement avec le reste du site
